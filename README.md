# Bidding Application – Backend

This is the Spring Boot backend for the Bidding Application.  
It handles product listings, bidding operations, user interactions,    and data management for the system.

---

## 🚀 Live Backend URL (Render – Free Plan)

https://bitdding-backend.onrender.com/api/products

(Wait for few seconds after opening — backend may take time to wake up.  
Once it loads successfully, connect with frontend.)

---

## ⚠️ Important Note (Render Free Plan)

This backend is hosted on **Render Free Plan**.

- Backend may sleep when inactive
- It may take **30–60 seconds** to start again
- Open backend URL once before using frontend

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/bidding-app-org/bitdding-backend.git
````

Navigate into project folder:

```bash
cd bitdding-backend
```

Run the application:

```bash
mvn spring-boot:run
```

Backend will run at:
(http://localhost:8080)
